# Getting Started

> [!NOTE]
> Once you start using this template, you should delete this file.

While not strictly required, basic knowledge of [Node.js](https://www.w3schools.com/nodejs/) will
help you understand this template more quickly.

1. **Generate your project:** Create a new repository from this template using GitHub's "Use this template" button above.

2. **Setup the environment:** Please follow the steps in [DEVELOPMENT.md](./DEVELOPMENT.md).

3. **Customize the build:** You'll likely want to tweak the configuration. Open the [build script](../scripts/build.ts) and adjust the variables to suit your needs.
